## Features

- **Intent Classification**: Uses TF-IDF + Logistic Regression to understand user queries  
- **Semantic Search**: Handles queries not covered in training data  
- **Professional UI**: Scrollable chat with user/bot message distinction and bot icon  
- **Expandable Data**: Easy to update `training_data.json` and `school_data.json`  
- **Logging**: Logs all conversations (optional)  
- **Responsive Design**: Works on desktop and mobile
